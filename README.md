# Computer Science Math
### The Cushman School — Willie Avendano

Applied mathematics through computation. Students write programs that explore statistics, forecasting, simulation, and algorithmic thinking — building real tools that solve real problems.

## Course Overview

- Descriptive and inferential statistics via code
- Monte Carlo simulation
- Forecasting models (time series, regression)
- Optimization and algorithmic decision-making
- Data visualization

## Tools & Languages
- Python (numpy, pandas, matplotlib, scikit-learn)
- Jupyter Notebooks
- GitHub
